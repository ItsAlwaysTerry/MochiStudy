## 5. 后续抓取命令思路

不要优先用 `yt-dlp` 抓合集页；优先用 B 站 API。

常用 API 形式：

```text
一轮 series:
https://api.bilibili.com/x/series/archives?mid=23630128&series_id=340933&only_normal=true&sort=desc&pn=1&ps=100

基础 season:
https://api.bilibili.com/x/polymer/web-space/seasons_archives_list?mid=23630128&season_id=2045&page_num=1&page_size=50
https://api.bilibili.com/x/polymer/web-space/seasons_archives_list?mid=23630128&season_id=2181&page_num=1&page_size=50

单个 BV 的 P 列表:
https://api.bilibili.com/x/player/pagelist?bvid=BV1D54y1m7Av
```

如果后续要把完整清单正式纳入项目，可以从这些 API 生成 JSON/Markdown。

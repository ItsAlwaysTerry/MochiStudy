# MochiStudy 用户手册 v3.2

## 一、核心思想

整个系统只做一件事：

```text
记录 -> 分析 -> 提示 -> 循环
```

完整闭环如下：

```text
AI 家教（学新东西）
  -> 输出 MOCHI-RECORD
Mochi 导入框（首页）
  -> 自动存档 + 分析弱点
Mochi 复习推荐（首页 / 复习页）
  -> 点击「复制材料」自动生成并复制复习包
复习 AI 私教（测验你）
  -> 输出 MOCHI-RECORD
Mochi 导入框（首页 / 复习页内联）
  -> 存档 + 更新权重
  -> 重新排序复习队列
```

你不需要手动选择要复习哪个知识点，也不需要手动导出任何东西来做日常复习。

系统会根据你的历史卡点、星级、距离上次学习的天数，自动算出今天最该复习什么，并在你点击时把材料复制好。

## 二、核心问题

### 是不是只用看学习网站的复习界面，等着它安排，然后点击复习，就会自动复制对应内容，再粘贴到复习 AI 就 OK 了？

是的，完全正确。这就是改版后的设计意图。

具体步骤：

1. 打开 Mochi 首页，看到「今日复习」卡片，里面显示今天推荐的知识点。
2. 点击「复制材料」，系统自动把这个知识点的完整复习包复制到剪贴板。复习包包含你的历史卡点、原题、套路。
3. 打开复习 AI 私教，粘贴材料，跟 AI 做复习测验。
4. 测验结束后，AI 输出 `MOCHI-RECORD`，复制它。
5. 回到 Mochi 首页的「今日复习」卡，此时已展开一个 textarea，粘贴记录并点击「导入复习结果」。
6. 完成，首页自动更新。

全程不需要手动导出任何东西。

## 三、导航结构

v3.2 之后，底部导航为 4 个入口：

```text
首页 / 学习 / 勋章 / 设置
```

「学习」页下有两个子 tab：

```text
复习队列：今天要复习哪些
学习档案：所有历史卡片
```

侧边栏同步为：

```text
首页 / 学习 / 勋章 / 赛季 / 设置
```

侧边栏不再有独立的「复习」和「档案」两个入口。

## 四、首页主操作区

首页从上到下包含以下区域。

### 1. 赛季徽章

如果当前有赛季，首页顶部会显示单行小标签，内容为赛季名和剩余天数。

点击徽章会跳转到赛季页。

### 2. 连续打卡横幅

连续打卡横幅只在有效学习日显示。

有记录时显示：

```text
连续 N 天 · 今天已导入 N 条
```

今天没有记录时显示：

```text
今天还没开始，打一张就够了
```

并提供「去导入」按钮。点击后，页面会自动滚动到导入框并 focus。

非有效学习日，也就是上学日，不显示该横幅。

### 3. 导入框

导入框是每天最重要的入口。

粘贴来源：AI 家教完成一道题后输出的 `MOCHI-RECORD`。

操作步骤：

1. 复制 AI 输出的整段记录。
2. 粘贴到首页导入框。
3. 点击「确认导入」。

导入后，系统会解析科目、知识点、星级、卡点、原题、套路，并存入学习档案。

上学日，也就是非假期时，导入框会替换成「农场休眠中」。如果今天仍想学习，可以点击「今天想学习」强制开启。

### 4. 今日复习卡

有历史记录后，首页会出现「今日复习」卡。

#### 状态 A：今天有推荐复习的知识点

卡片会显示：

```text
科目 · 知识点名
```

卡点默认折叠。点击「还记得吗？」可以展开，用于主动回忆。

按钮包括：

- 「复制材料」
- 「看全部」

点击「复制材料」后：

1. 显示「材料已复制，先回想 20 秒」。
2. 展开 textarea。
3. 将复习 AI 输出的 `MOCHI-RECORD` 粘贴进去。
4. 点击「导入复习结果」。
5. 成功后显示反馈消息和「继续」按钮。
6. 点击「继续」后清除当前状态。

#### 状态 B：今天无推荐

如果近期复习过的知识点都在巩固中，卡片会显示：

```text
近期复习过的知识点都在巩固中，今天可以专心做新题。
```

如果有即将到期的知识点，还会显示：

```text
下一个到期：[知识点名] · N 天后
```

此时按钮为「看复习页」，点击后跳转到复习队列完整列表。

### 5. 迷你农场

迷你农场包含三块地：

- 数学
- 物理
- 化学

每块地会显示当前生长阶段和进度，例如：

```text
N / 15
```

成熟后会出现「收」按钮。

### 6. 专注计时器

专注计时器默认折叠。

点击「开始专注」后展开番茄钟。平时折叠时不占首页空间。

## 五、学习 -> 复习队列

复习队列是首页「看全部」按钮跳转的地方，也是导航「学习」下的「复习队列」子 tab。

它和首页复习卡的区别是：

- 首页只显示今天推荐的第一个知识点，是快捷入口。
- 复习队列显示全部待处理知识点，并按优先级排序。

列表中的每一行包含：

- 科目标签
- 知识点名
- 推荐原因，也就是为什么今天推荐
- 今日建议条目的绿色左边框
- 「今日」绿色徽章

操作流程：

1. 点击某个知识点的「开始」按钮。
2. 系统自动把该知识点的复习包复制到剪贴板。
3. 复制成功后展开导入面板。
4. 粘贴复习 AI 输出的结果。
5. 点击导入。
6. 完成复习闭环。

复习队列底部还有一个折叠区：

```text
冷却 / 已稳定
```

这里展示近期不需要处理的知识点，可以展开查看。

复习队列的操作流程和首页完全一样，只是可以手动选择任意知识点复习，不局限于今日建议的第一个。

## 六、学习 -> 学习档案

学习档案有三个 tab：

- 数学
- 物理
- 化学

每个知识点展开后，会显示所有历史卡片。

每张卡片有三面：

- 正面：日期、星级、卡点，默认显示。
- 今日套路：3 步套路，点击切换。
- 看原题：当时的原题描述，点击切换。

右上角有「导出档案」按钮，点击后弹出导出面板。

### 导出格式

| 格式 | 内容 | 用途 |
| --- | --- | --- |
| 摘要 | 每个知识点的概况：学了几次、进步轨迹、历史卡点 | 给 AI 做整体学习分析 |
| 详细记录 | 每张卡片的完整内容：日期、星级、题数、卡点、原题、套路 | 给 AI 针对卡点出题或制定复习计划 |
| AI 复习包 | 所有待复习知识点的复习素材，批量版 | 给复习 AI 一次生成多个知识点的测验 |
| JSON | 原始结构化数据 | 开发或自动化处理 |

可以选择的导出范围：

- 全部
- 数学
- 物理
- 化学

说明：「AI 复习包」是批量导出，适合偶尔让 AI 做全面诊断。日常逐条复习直接用首页或复习队列的「复制材料」按钮。这个按钮会按单个知识点精准生成材料，不需要到导出档案里操作。

## 七、两类 AI 交互

### 类型 A：学新东西（AI 家教）

流程：

```text
你：「我要学[知识点]」
AI：出题 -> 讲解 -> 输出 MOCHI-RECORD
你：复制 MOCHI-RECORD -> 粘贴到 Mochi 首页导入框 -> 确认导入
```

AI 输出的 `MOCHI-RECORD` 包含：

- 科目
- 知识点
- 卡点
- 原题
- 套路
- 星级

这种记录里的「学习来源」是：

```text
课堂 / 新学
```

### 类型 B：复习测验（复习 AI 私教）

流程：

```text
Mochi：自动生成复习包
你：复制复习包 -> 粘贴给复习 AI
AI：先测你，不直接给答案
你：尝试作答
AI：根据情况给提示
AI：复习完成后输出 MOCHI-RECORD
你：复制 MOCHI-RECORD -> 粘贴到 Mochi 复习卡 textarea -> 导入
```

复习包包含：

- 你的历史卡点
- 原题
- 套路
- 给 AI 的复习指令

这种记录里的「学习来源」是：

```text
复习测验
```

复习记录还会包含额外字段，例如：

- 复习结果
- 错误类型
- 卡住步骤

系统会记录这些字段，并影响该知识点下次的复习优先级。

## 八、系统如何决定今天复习什么

系统会综合以下因素给每个知识点打分，分高的优先推荐：

- 低星次数：1 星、2 星记录越多，分越高。
- 重复出现的相同卡点：同一个卡点反复出现，分越高。
- 距离上次学习的天数：越久没碰，分越高。
- 复习后仍未独立做对的次数：多次复习仍卡住，分越高。
- 是否到了下次复习日期：基于上次星级计算冷却期。

3 星记录会拉长冷却期，让这个知识点暂时降权。

1 星记录会加快下次出现。

## 九、其他页面

### 勋章页

触发勋章检查的动作包括：

- 导入记录
- 完成番茄钟
- 收获农场

攒到一定数量的小勋章或大勋章后，可以兑换抽奖次数。

有可用抽奖次数时，导航按钮右上角会出现红色数字。

点击抽奖后，转盘会抽取奖励，例如妈妈做好吃的、看综艺、做深蹲等。

### 赛季页

管理员在 `?admin=1` 开启赛季后，赛季页会显示：

- 倒计时
- 称号
- 热力图
- 趋势图

无赛季时，赛季页会显示累计学习统计：

- 记录数
- 学习天数
- 专注小时
- 三科数量

### 设置页

设置页包含：

- 数据备份和恢复，使用 JSON 文件。
- 番茄钟提示音选择。
- 学年日历配置，也就是哪些日期是假期。

## 十、常见操作速查

| 我想做什么 | 去哪里 |
| --- | --- |
| 导入今天做的题 | 首页 -> 导入框 |
| 做今天推荐的复习 | 首页 -> 今日复习卡 -> 复制材料 |
| 查看所有待复习的题 | 学习 -> 复习队列 |
| 翻看历史学习记录 | 学习 -> 学习档案 |
| 导出给 AI 分析 | 学习 -> 学习档案 -> 导出档案 |
| 收获农场 | 首页 -> 迷你农场 -> 「收」按钮 |
| 抽奖 | 勋章页，导航有红色徽章时 |
| 开启或结束赛季 | `?admin=1` 管理后台 |
| 备份或恢复数据 | 设置页 |

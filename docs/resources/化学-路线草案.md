# 化学一轮基础路线草案

## 选材口径

- 学生当前化学约 20 多分，本路线按“从零建体系”处理，不按高考冲刺处理。
- 主骨架来自 `数化-初筛.md` 的化学链接 7：原子结构、物质的量、钠/硫/氮元素化合物、有机入门。
- 补充来自化学链接 5 的 40 分以下基础视频：氧化还原、无机非金属材料、有机官能团、取代反应。
- 已跳过：所有“最后十课”“高考冲刺”“综合大题”类 2-4 小时视频；盐类水解、电化学、有机推断等不作为 20 分学生第一轮地基。
- MochiStudy 知识点只能落到：原子结构、化学键、氧化还原反应、化学反应、化学平衡、电化学、有机化学。本路线不新增 nodeLabel。

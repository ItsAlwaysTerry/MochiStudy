# 暑假物理视频例题收集

这个文件夹用来收集 B 站视频里的例题截图，后续整理进 MochiStudy 的暑假任务区。

## 怎么用

打开 `例题收件箱.md`，里面已经按视频分好了区域，每个区域都有 B 站链接。

你只需要点击对应视频链接，找到例题后，把截图直接粘到这个视频区域下面。最多在截图前写一个时间点，比如 `08:20`；不写也可以。

## 图片放哪里

截图统一放在 `screenshots/` 目录里，markdown 里使用 `screenshots/xxx.png` 相对路径。这样 push 到 GitHub 后，学生 pull 下来也能看到原图。

`例题收件箱.md` 顶部已经写了 Typora 配置：

```yaml
typora-copy-images-to: screenshots
typora-root-url: .
```

以后在这个文件里粘贴图片，Typora 会优先把图片复制到旁边的 `screenshots/`，不要再放到 `C:\Users\...\Typora\typora-user-images`。

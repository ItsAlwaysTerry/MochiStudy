# 例题收件箱

使用方式：打开对应视频区域，点击视频链接去 B 站找例题，然后把截图直接粘在这个区域下面。

你不用写编号，不用复制截图文件名，也不用判断题型。最多在图片前写一下时间点，例如：

```text
08:20
```

如果懒得写时间点也没关系，直接粘图。
